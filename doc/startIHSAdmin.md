## startIHSAdmin

The startIHSAdmin tool starts the IBM HTTP Administration Server on the current system.

### Syntax

#### Linux

```Shell
$ sudo startIHSAdmin.sh
```

#### Windows

```Shell
> startIHSAdmin.ps1
```

### Options

None.

### Example

```Shell
$ sudo startIHSAdmin.sh
Starting IHS Admin server...                                 SUCCESS
```
