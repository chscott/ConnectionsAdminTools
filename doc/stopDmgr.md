## stopDmgr
The stopDmgr command stops the WAS Deployment Manager on the current system.

### Syntax

#### Linux

```Shell
$ sudo stopDmgr.sh
```

#### Windows

```Shell
> stopDmgr.ps1
```

### Options

None.

### Example

```Shell
$ sudo stopDmgr.sh
Stopping server dmgr in profile dmgr...                      SUCCESS
```
