## startDmgr
The startDmgr command starts the WAS Deployment Manager on the current system.

### Syntax

#### Linux

```Shell
$ sudo startDmgr.sh
```

#### Windows

```Shell
> startDmgr.ps1
```

### Options

None.

### Example

```Shell
$ sudo startDmgr.sh
Starting server dmgr in profile dmgr...                      SUCCESS
```
