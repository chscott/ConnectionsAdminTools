## getInstalledFixes

The getInstalledFixes tool returns a formatted list of iFixes applied to your Connections Blue environment.

### Syntax

```Bash
$ sudo getInstalledFixes.sh
```
### Options

None.

### Example

```Bash
$ sudo getInstalledFixes.sh
```
