## syncUsers

The syncUsers tool is a simple wrapper around the TDI sync_all_dns.sh tool, which initiates LDAP-to-Profiles synchronization.

### Syntax

#### Linux
```
$ sudo syncUsers.sh
```

#### Windows
```
> syncUsers.ps1
```

### Options

None.

### Example

```
$ sudo syncUsers.sh
Synchronizing Profiles with LDAP...                          SUCCESS
```
